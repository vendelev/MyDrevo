# Автоматическое исправление кода и стиля (Rector и PHPCBF)

Запустите агента `php-auto-fixer` для автоматического исправления кода и стиля:

```text
Используйте Task tool (switch_mode) с subagent_type="php-auto-fixer"
```

Агент выполнит:

1. Rector - автоматическое улучшение кода
2. PHPCBF - автоматическое исправление стиля кода
