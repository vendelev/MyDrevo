# Модуль Core

## Назначение

Модуль **Core** обеспечивает базовую инфраструктуру приложения и интеграцию
модульной структуры `backend/src/*` с Laravel.

## Архитектурная роль

- регистрирует web-маршруты приложения через `RouteServiceProvider`;
- подключает файл маршрутов `routes/web.php`, где находятся web-интерфейсы
  модулей (на текущем этапе — Auth).

## Компоненты

### core.web.router

Обеспечивает подключение web-маршрутов приложения и связывает Core с
web-интерфейсами модулей.
