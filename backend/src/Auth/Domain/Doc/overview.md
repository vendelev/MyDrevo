# Модуль Auth

## Назначение

Модуль **Auth** реализует базовую web-аутентификацию пользователей проекта:
регистрацию, вход, выход и доступ к dashboard.

## Архитектурная роль

- принимает HTTP-запросы через контроллеры;
- выполняет бизнес-логику регистрации и входа через UseCase и CQRS-команды;
- взаимодействует с таблицей `gen_user` в SQLite;
- управляет сессиями через Laravel guard `web`.

## Компоненты

### auth.web

Web-контур модуля: контроллеры и Blade-представления, которые обслуживают
формы регистрации/входа и страницу dashboard.

### auth.user.service

UseCase и CQRS-команды/запросы, отвечающие за регистрацию и проверку учетных
данных пользователя, а также за завершение сессии.

### db.gen_user

Таблица пользователей, которая хранит учетные данные и профильные поля для
web-аутентификации.
